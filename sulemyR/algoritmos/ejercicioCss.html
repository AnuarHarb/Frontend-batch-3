Ejercicio Presupuesto

Inicio(ingreso1, ingreso2, gasto1, gasto2, gasto3)
    leer ingreso1
    leer ingreso2
    leer gasto1
    leer gasto2
    leer gasto3

    presupuesto = (ingreso1 + ingreso2) - (gasto1 + gasto2 + gasto3)

    Escribe presupuesto
Fin

Ejercicio Herencia de nietos

Inicio(jose, juan, herencia)
    leer jose
    leer juan
    leer herencia

    herenciaIndividual = (herencia / 2)
    valorJose = jose + herenciaIndividual
    valorJuan = juan + herenciaIndividual

    valorTotal = valorJose + valorJuan

    Escribe valorJose
    Escribe valorJuan
    Escribe valorTotal
Fin
